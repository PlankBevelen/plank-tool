# Plank Tool

A powerful toolbox for developers, built with React, TypeScript, and Vite.

## Features

- **Text Tools**: Text manipulation, conversion, and formatting.
- **Image Tools**: Compression, format conversion, QR code generation.
- **JSON Tools**: Formatting, validation, and conversion.
- **Time Tools**: Time conversion and calculation.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- React Router
- Framer Motion
